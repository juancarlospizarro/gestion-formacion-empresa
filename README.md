# Aplicación Web de Gestión de Prácticas en Empresa

## 🇪🇸 Descripción
Este proyecto es una aplicación web desarrollada con **Spring Boot** que permite la gestión y la consulta del periodo de formación (prácticas) de alumnos en empresas, tanto por parte del profesorado como de los directivos. El objetivo principal es facilitar el seguimiento de las mismas por parte del centro educativo.

### Tecnologías:
* **Backend:** Java 21, Spring Boot (Spring Security, Spring Validation, Spring MVC, Spring Mail, Spring Data JPA)
* **Gestor de Dependencias:** Maven
* **Frontend:** Spring Thymeleaf, Bootstrap
* **Base de Datos:** MySQL
* **Contenedores:** Docker & Docker Compose (No implementado todavía)

# Web Application for Internship Management

## 🇬🇧 Description
This project is a web application built with **Spring Boot** designed to manage and monitor student internships in companies. It allows both teachers and academic directors to oversee the training period, track progress and maintain a direct communication channel with host companies. The primary goal is to streamline the internship follow-up process for educational institutions.

### Tech Stack:
* **Backend:** Java 21, Spring Boot (Spring Security, Spring Validation, Spring MVC, Spring Mail, Spring Data JPA)
* **Dependency Management:** Maven
* **Frontend:** Spring Thymeleaf, Bootstrap
* **Database:** MySQL
* **Containerization:** Docker & Docker Compose (Not implemented yet)
